3. **Figma** - For mockups and compositions
4. **Shotsnapp** - Free browser-based tool

### Enhancement Checklist:
- [ ] Use device frames (iPhone, MacBook mockup)
- [ ] Add subtle shadows/depth
- [ ] Use consistent font for annotations (Inter, SF Pro)
- [ ] Keep text overlays minimal and readable
- [ ] Use brand colors (Emerald #10b981)
- [ ] Ensure high DPI (2x or 3x resolution)
- [ ] Optimize file size (< 500KB per image)
- [ ] Use PNG for screenshots (lossless)

### Annotation Style Guide:
- **Arrows:** Emerald green (#10b981), 3px width
- **Text:** White on semi-transparent dark background
- **Font:** SF Pro Display (iOS) or Inter (Web)
- **Size:** 18-24px for main text, 14-16px for captions
- **Highlight boxes:** Emerald border, 10% fill opacity
